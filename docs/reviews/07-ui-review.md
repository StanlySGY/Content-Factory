## 3. 审查清单

- [ ] 页面树与信息架构一致，导航深度可控
- [ ] 全局布局与响应式策略合理
- [ ] 通用组件覆盖核心场景，职责清晰
- [ ] 交互设计遵循状态可见、可控、防错、渐进披露
- [ ] 核心页面齐全：Dashboard、内容中心、文章编辑页、素材中心、知识库、Agent 管理、MCP 管理、公众号工作台
- [ ] 各页面结构、交互与工作流 / Agent / MCP 状态一致
- [ ] 高风险动作采用阻断式确认
- [ ] 空状态与错误状态设计完整
- [ ] 可访问性与性能要求满足（对比度、键盘、骨架屏、reduced-motion）
- [ ] 前端不承载业务规则，不直连 Agent / MCP / Skill / 插件内部实现
